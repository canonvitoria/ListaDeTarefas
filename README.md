## Funcionalidades

- [x] **Adicionar Tarefas:** Permite inserir novas tarefas via input de texto.
- [x] **Remover Tarefas:** Botão dedicado para excluir itens da lista.
- [x] **Persistência de Dados:** As tarefas são salvas no `localStorage` do navegador, ou seja, elas não somem se você fechar a aba ou recarregar a página.
- [x] **Validação:** Impede a criação de tarefas vazias.
- [x] **Interatividade:** Adição via tecla `Enter` ou clique no botão.
- [x] **Design Responsivo:** Interface limpa e adaptável.

---

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização, Flexbox e efeitos de transição.
* **JavaScript (Vanilla):** Lógica de manipulação do DOM e LocalStorage (JSON).
